# Setup & Run

## 1. Install Ollama
Make sure Ollama is installed on your system.

## 2. Pull the model
```bash
ollama pull mistral:7b-instruct
```
## 3. Start ollama
```bash
ollama serve
```

## Run python script
```bash
python main.py

```

## Start web client
```bash
cd web-client
python -m http.server 8000
```

## See the web client
http://localhost:8000
